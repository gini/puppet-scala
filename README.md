Scala Puppet Module
====================

[![Build Status](https://secure.travis-ci.org/gini/puppet-scala.png)](http://travis-ci.org/gini/puppet-scala)

Overview
--------

Install the Scala programming language from the official project site.


Usage
-----

Example:

    class { 'scala':
      version => '2.10.2',
    }
